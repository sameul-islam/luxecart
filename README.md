# LuxeCart - Modern E-Commerce Website

LuxeCart is a modern, responsive, and high-performance e-commerce website built with Next.js, TypeScript, and Tailwind CSS.  
It is designed with a clean layout, smooth animations, and a professional structure suitable for real-world online shopping platforms.

---

## Features

- Dynamic navigation with category-based routing  
- Smart search overlay with real-time results  
- Product filtering by category, size, color, and price  
- Fully responsive layout across all devices  
- Modern About page with professional design and large images  
- Built using the latest versions of Next.js, TypeScript, and Tailwind CSS  

---

## Tech Stack

- **Next.js:** 16.0.1  
- **React:** 19.2.0  
- **TypeScript:** ^5  
- **Tailwind CSS:** ^4  
- **React Icons:** ^5.5.0  
- **ESLint, PostCSS** for clean code and optimization
